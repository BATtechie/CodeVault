# CodeVault CSS Design System Guide\n\n## Overview\n\nCodeVault uses a unified, production-ready design system with consistent spacing, typography, colors, and components. This guide ensures all CSS follows production standards.\n\n---\n\n## Color System\n\n### Primary Colors (from :root in index.css)\n\n```css\n--accent: #19446c              /* Primary blue - buttons, links, accents */\n--accent-strong: #8760ff       /* Purple accent - gradients, highlights */\n```\n\n### Surface & Background\n\n```css\n--surface: #f7f7fb             /* Main background light */\n--surface-alt: #eff2f8         /* Alternative surface for subtle contrast */\n```\n\n### Text Colors\n\n```css\n--text-primary: #101729        /* Main text, headings */\n--text-secondary: #48536c      /* Secondary text, descriptions */\n--text-muted: #69758f          /* Muted text, hints */\n--text-emphasis: #19446c       /* Emphasized text (same as accent) */\n--text-inverse: #f6f9ff        /* Text on dark backgrounds */\n```\n\n### Borders & Dividers\n\n```css\n--border-soft: rgba(19, 28, 52, 0.1)      /* Light borders, cards */\n--border-strong: rgba(255, 255, 255, 0.12) /* Strong borders, separators */\n```\n\n### Shadows\n\n```css\n--shadow-card: 0 24px 50px rgba(17, 24, 39, 0.08)    /* Card shadows */\n--shadow-button: 0 16px 30px rgba(25, 68, 108, 0.22)  /* Button shadows */\n```\n\n---\n\n## Typography\n\n### Font Families\n\n```css\n--font-body: 'Space Grotesk', 'Avenir Next', 'Segoe UI', sans-serif\n--font-mono: 'IBM Plex Mono', 'SFMono-Regular', Consolas, monospace\n```\n\n### Font Sizes (Typographic Scale)\n\n**Headings:**\n- H1: `clamp(2.8rem, 8vw, 5.7rem)` (Hero titles)\n- H2: `clamp(2rem, 5vw, 3.6rem)` (Page titles)\n- H3: `clamp(1.35rem, 1.1rem + 1vw, 2rem)` (Section titles)\n- H4: `1.1rem`\n\n**Body Text:**\n- Large: `1rem`\n- Regular: `0.95rem`\n- Small: `0.92rem`\n- Tiny: `0.8rem`\n\n**Monospace (Code):**\n- `0.82rem` (line numbers)\n- `0.88rem` (inline code)\n- `0.9rem` (code blocks)\n\n### Font Weights\n\n```css\nfont-weight: 400;  /* Regular - body text */\nfont-weight: 500;  /* Medium - subtle emphasis */\nfont-weight: 700;  /* Bold - headings, buttons, labels */\n```\n\n### Line Heights\n\n```css\nline-height: 1.05;  /* Tight (headings) */\nline-height: 1.3;   /* Normal headings */\nline-height: 1.6;   /* Comfortable (descriptions) */\nline-height: 1.65;  /* Extended (paragraphs) */\nline-height: 1.7;   /* Loose (body content) */\nline-height: 1.75;  /* Very loose (long-form text) */\n```\n\n### Letter Spacing\n\n```css\nletter-spacing: -0.04em;  /* Headings (negative) */\nletter-spacing: 0.02em;   /* Normal */\nletter-spacing: 0.06em;   /* Labels */\nletter-spacing: 0.08em;   /* Uppercase */\nletter-spacing: 0.12em;   /* Small caps */\n```\n\n---\n\n## Spacing System\n\n### Base Unit: 1rem = 16px\n\n```css\n/* Core spacing scale */\n0.35rem  = 5.6px   (icon gap)\n0.4rem   = 6.4px   (badge gap)\n0.45rem  = 7.2px   (small gap)\n0.5rem   = 8px     (label gap)\n0.65rem  = 10.4px  (label/icon gap)\n0.75rem  = 12px    (input padding)\n0.8rem   = 12.8px  (padding, button text)\n0.85rem  = 13.6px  (spacing, button padding)\n0.9rem   = 14.4px  (padding)\n0.95rem  = 15.2px  (spacing, input label)\n1rem     = 16px    (base unit)\n1.1rem   = 17.6px  (spacing)\n1.15rem  = 18.4px  (button padding)\n1.25rem  = 20px    (container padding, gap)\n```\n\n### Container Widths\n\n```css\nwidth: min(1180px, calc(100% - 1.5rem))  /* Standard container */\nmax-width: 56rem                         /* Content max-width */\nmax-width: 42rem                         /* Narrow content */\n```\n\n---\n\n## Border Radius\n\n```css\nborder-radius: 0.85rem;   /* Inputs, small elements */\nborder-radius: 1rem;      /* Forms, small containers */\nborder-radius: 1.25rem;   /* Code blocks */\nborder-radius: 1.5rem;    /* Cards, modals */\nborder-radius: 1.6rem;    /* Dashboard sections */\nborder-radius: 1.75rem;   /* Editor sections */\nborder-radius: 2rem;      /* Large panels */\nborder-radius: 999px;     /* Fully rounded (pills, buttons) */\n```\n\n---\n\n## Transitions & Animations\n\n### Timing\n\n```css\ntransition: 160ms ease;    /* Default smooth transition */\ntransition: 160ms ease-in; /* Ease in animation */\n```\n\n### Common Animations\n\n```css\n/* Hover effects */\ntransform: translateY(-1px);  /* Subtle lift */\ntransform: translateY(-2px);  /* Stronger lift */\n\n/* Box shadows */\nbox-shadow: 0 12px 20px rgba(28, 39, 74, 0.08);    /* Light hover */\nbox-shadow: 0 16px 40px rgba(82, 72, 179, 0.25);   /* Strong hover */\n```\n\n---\n\n## Component Patterns\n\n### Buttons\n\n```css\n/* Primary Button */\n.btn-primary {\n  background: linear-gradient(135deg, var(--accent), var(--accent-strong));\n  color: white;\n  padding: 0.9rem 1.1rem;\n  border-radius: 999px;\n  border: none;\n  font-weight: 700;\n  cursor: pointer;\n  box-shadow: var(--shadow-button);\n  transition: 160ms ease;\n}\n\n.btn-primary:hover {\n  transform: translateY(-1px);\n}\n\n/* Ghost Button */\n.btn-ghost {\n  background: white;\n  color: var(--text-primary);\n  border: 1px solid var(--border-soft);\n  padding: 0.8rem 1.1rem;\n  border-radius: 999px;\n  font: inherit;\n  cursor: pointer;\n  transition: 160ms ease;\n}\n\n.btn-ghost:hover {\n  background: rgba(40, 68, 135, 0.08);\n  transform: translateY(-1px);\n}\n```\n\n### Cards\n\n```css\n/* Standard Card */\n.card {\n  padding: 1.25rem;\n  border-radius: 1.5rem;\n  border: 1px solid var(--border-soft);\n  background: rgba(255, 255, 255, 0.9);\n  box-shadow: var(--shadow-card);\n  transition: 160ms ease;\n}\n\n.card:hover {\n  transform: translateY(-2px);\n  box-shadow: 0 16px 40px rgba(28, 39, 74, 0.1);\n}\n```\n\n### Forms\n\n```css\n/* Input Elements */\ninput,\nselect,\ntextarea {\n  color: var(--text-primary);\n  background: white;\n  border: 1px solid var(--border-soft);\n  border-radius: 0.85rem;\n  padding: 0.75rem 0.95rem;\n  font: inherit;\n  transition: 160ms ease;\n}\n\ninput:focus,\nselect:focus,\ntextarea:focus {\n  outline: none;\n  border-color: var(--accent-strong);\n  box-shadow: 0 0 0 2px rgba(135, 96, 255, 0.1);\n}\n\n/* Labels */\nlabel {\n  display: grid;\n  gap: 0.5rem;\n  font-weight: 700;\n  color: var(--text-primary);\n  font-size: 0.92rem;\n}\n```\n\n### Badges & Pills\n\n```css\n/* Badge */\n.badge {\n  display: inline-flex;\n  align-items: center;\n  gap: 0.35rem;\n  padding: 0.4rem 0.75rem;\n  border-radius: 999px;\n  background: rgba(40, 68, 135, 0.08);\n  color: var(--text-emphasis);\n  font-size: 0.82rem;\n  font-weight: 700;\n}\n```\n\n---\n\n## Responsive Design\n\n### Breakpoints\n\n```css\n/* Mobile First */\n@media (max-width: 640px) {\n  /* Mobile styles */\n}\n\n@media (min-width: 641px) and (max-width: 900px) {\n  /* Tablet styles */\n}\n\n@media (min-width: 901px) {\n  /* Desktop styles */\n}\n```\n\n### Fluid Typography (clamp)\n\nUse `clamp()` for responsive sizing:\n\n```css\nfont-size: clamp(1rem, 2vw, 1.5rem);\n/* min: 16px, preferred: 2% of viewport, max: 24px */\n\npadding: clamp(1rem, 4vw, 2rem);\n/* min: 16px, preferred: 4% of viewport, max: 32px */\n```\n\n### Grid Layouts\n\n```css\n/* Auto-responsive grid */\ndisplay: grid;\ngrid-template-columns: repeat(auto-fit, minmax(220px, 1fr));\ngap: 1.25rem;\n\n/* Fixed responsive grid */\n@media (min-width: 900px) {\n  grid-template-columns: repeat(3, minmax(0, 1fr));\n}\n```\n\n---\n\n## Accessibility\n\n### Focus States\n\n```css\n/* Always visible focus indicators */\nbutton:focus,\ninput:focus,\na:focus {\n  outline: 2px solid var(--accent-strong);\n  outline-offset: 2px;\n}\n```\n\n### High Contrast Mode\n\n```css\n@media (prefers-contrast: more) {\n  /* Increase contrast */\n  --text-primary: #000;\n  --border-soft: rgba(0, 0, 0, 0.2);\n}\n```\n\n### Reduced Motion\n\n```css\n@media (prefers-reduced-motion: reduce) {\n  * {\n    animation-duration: 0.01ms !important;\n    animation-iteration-count: 1 !important;\n    transition-duration: 0.01ms !important;\n  }\n}\n```\n\n---\n\n## Production Best Practices\n\n### тЬЕ DO\n- Use CSS custom properties (--variable-name)\n- Use rem units for scalability\n- Use clamp() for fluid typography\n- Test on multiple devices\n- Use semantic HTML with proper heading hierarchy\n- Optimize images and minimize CSS\n- Use transitions for smooth interactions\n- Test with accessibility tools\n\n### тЭМ DON'T\n- Use inline styles\n- Use px units for spacing (except shadows)\n- Use hard-coded colors (use CSS variables)\n- Add !important rules\n- Nest CSS selectors unnecessarily\n- Use @media queries without mobile-first approach\n- Forget to test responsive design\n- Use deprecated CSS properties\n\n---\n\n## CSS Maintenance\n\n### File Organization\n\n```\nsrc/\nтФЬтФАтФА index.css           # Global styles, variables\nтФЬтФАтФА App.css             # App layout\nтФЬтФАтФА components/\nтФВ   тФЬтФАтФА Navbar.css\nтФВ   тФЬтФАтФА Footer.css\nтФВ   тФЬтФАтФА CodeBlock.css\nтФВ   тФЬтФАтФА SnippetCard.css\nтФВ   тФФтФАтФА SnippetEditor.css\nтФФтФАтФА pages/\n    тФЬтФАтФА Home.css\n    тФЬтФАтФА LandingPage.css\n    тФЬтФАтФА SignIn.css\n    тФЬтФАтФА Dashboard.css\n    тФЬтФАтФА Profile.css\n    тФФтФАтФА Snippets.css\n```\n\n### Naming Convention\n\n- Use BEM (Block Element Modifier) pattern\n- Example: `.card__header`, `.card__title`, `.card--active`\n- Keep selectors specific but not overly deep\n- Use classes, avoid IDs for styling\n\n### Code Quality\n\n- Run `npm run lint` before committing\n- Keep CSS files under 50KB (minified)\n- Remove unused CSS regularly\n- Use CSS variables for themeable values\n- Document complex selectors with comments\n\n---\n\n## Examples\n\n### Example 1: Hero
