Security Headers\n\n```javascript\n// Already configured in src/index.js\napp.use(\n  helmet({\n    crossOriginResourcePolicy: false,\n  }),\n);\n\n// Headers applied:\n// - X-Content-Type-Options: nosniff\n// - X-Frame-Options: DENY\n// - X-XSS-Protection: 1; mode=block\n// - Strict-Transport-Security: (HSTS)\n// - Content-Security-Policy\n```\n\n### CORS Configuration\n\n```javascript\n// Configured with origin whitelist\napp.use(\n  cors({\n    origin: function(origin, callback) {\n      // Allows requests from whitelisted origins only\n    },\n    credentials: true,\n    methods: ['GET', 'POST', 'PUT', 'PATCH', 'DELETE', 'OPTIONS'],\n    allowedHeaders: ['Content-Type', 'Authorization', 'x-csrf-token'],\n  }),\n);\n\n// ✅ In production:\n// - Only whitelisted origins allowed\n// - Credentials included\n// - Specific HTTP methods allowed\n```\n\n### CSRF Protection\n\n```javascript\n// Double-submit cookie pattern implemented\napp.use(generateCsrfTokenMiddleware); // Generates on every request\napp.use(csrfMiddleware);              // Validates on state-changing ops\n\n// Protected endpoints:\n// - POST /auth/signup, /auth/login\n// - POST /snippets (create)\n// - PUT/PATCH /snippets/:id (update)\n// - DELETE /snippets/:id (delete)\n// - POST /teams (create)\n// - All state-changing operations\n```\n\n### Cookie Configuration\n\n```javascript\n// Secure flags applied to all cookies:\n// - Secure: Only transmitted over HTTPS\n// - HttpOnly: Not accessible via JavaScript\n// - SameSite: Strict (CSRF protection)\n\nres.cookie('sessionId', token, {\n  secure: true,       // Production only\n  httpOnly: true,     // No JS access\n  sameSite: 'strict', // CSRF protection\n  maxAge: 7 * 24 * 60 * 60 * 1000, // 7 days\n});\n```\n\n---\n\n## Rate src/index.js\napp.use((err, req, res, next) => {\n  // Production mode: Hide implementation details\n  // Development mode: Include full stack trace\n  \n  if (process.env.NODE_ENV === 'production') {\n    // Return generic message\n    return res.status(500).json({\n      success: false,\n      message: 'Internal server error',\n    });\n  } else {\n    // Return detailed error\n    return res.status(500).json({\n      success: false,\n      message: err.message,\n      stack: err.stack,\n    });\n  }\n});\n```\n\n### HTTP Status Codes\n\n- 200: Success\n- 201: Created\n- 400: Bad
